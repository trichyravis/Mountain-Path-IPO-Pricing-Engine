# ⛰ The Mountain Path — Primary Market Equity Pricing Model
### Prof. V. Ravichandran | themountainpathacademy.com

A comprehensive, interactive Streamlit application for analysing IPO equity pricing
using the **Book Building** and **Fixed Price** methods.

---

## 🚀 Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run the app
streamlit run app.py
```

The app opens at **http://localhost:8501**

---

## 📊 Features

| Tab | Content |
|-----|---------|
| **Dashboard** | Executive summary with 10 live metrics, 4 interactive charts |
| **Book Building** | Demand schedule, cut-off price discovery, category allocation, proceeds |
| **Fixed Price** | Subscription analysis, allotment & refund breakdown, proceeds statement |
| **Comparison** | Quantitative scorecard, qualitative assessment, radar chart |
| **Formulas** | 10 master formulas with live computed results |
| **Learn** | Q&A study guide, SEBI regulatory framework reference |

---

## 🎛️ Sidebar Controls

All inputs are live — the entire model recalculates instantly:

- **Company**: Name, pre-IPO shares, IPO shares, face value
- **Book Building**: Floor price, cap price
- **Fixed Price**: Fixed offer price
- **Costs**: Underwriting %, legal, marketing, listing fees
- **Allocation**: QIB/NII/RII reservation percentages
